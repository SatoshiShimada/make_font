# OSリポジトリ用のフォントデータ

自作OS用のフォントデータ

現在16pixel X 16pixelのデータのみ

## font16x16.txt

'@'と'.'でビットマップフォントを表現

'@'は背景

'.'は点灯する部分

make_fontを実行するとfont16x16.txtから

C言語のソースプログラムに変換して

font_data.cに出力する

unsigned short 型の配列
